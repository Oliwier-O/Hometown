#  Hometown Project – Sosnowiec

This is a simple static website about my hometown – Sosnowiec (Poland).  
It was created as a practice project to improve my HTML and CSS skills.

##  About the project
The website includes:
- Top activities in Sosnowiec
- A guide section with a personal profile
- A location map
- Navigation bar with smooth scrolling

##  Learning process
This project was built while learning front-end basics.  
After some time away from coding, I came back and rebuilt my fundamentals.

I also used AI assistance to help me understand layout issues, Flexbox behavior, and debugging CSS problems.

##  Tech used
- HTML5
- CSS3 (Flexbox, responsive design basics)

##  Purpose
This project is purely educational and part of my learning journey in web development.
